Frequently Asked Questions

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects (or nodes) where each object corresponds to part of the document (e.g., elements, attributes, or text). Using the DOM, JavaScript can interact with and manipulate the content, structure, and styling of web pages dynamically.

How do you select an element from the DOM?

By ID: let element = document.getElementById('elementId');
By class: let elements = document.getElementsByClassName('className');
By tag name: let elements = document.getElementsByTagName('div');
Using querySelector(): let element = document.querySelector('.className');
Using querySelectorAll(): let elements = document.querySelectorAll('.className');

What is event delegation in the context of the DOM, and how is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events triggered by its child elements. Instead of adding event listeners to each child individually, you listen for an event on the parent and check which child triggered the event using the event.target.

Why it's useful:
1. Performance: It reduces the number of event listeners required, which is especially useful when dealing with a large number of elements.
2. Dynamic content: It works well for elements added to the DOM after the initial event listeners have been set up.

How do you manipulate an element's attributes and styles using the DOM?

To set or get an attribute:
let element = document.getElementById('myElement');
element.setAttribute('src', 'image.jpg');
let srcValue = element.getAttribute('src');


To change an element's inline CSS styles:
let element = document.getElementById('myElement');
element.style.color = 'red';
element.style.backgroundColor = 'blue';
element.style.fontSize = '20px';
